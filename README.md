# Math Re:Start

中高生向けの数学おさらいサイトです。整数の計算ルールから平方根、文字式、分配法則、方程式、関数、図形・データまでを、短い説明・動かせる図解・自動生成されるステップ問題で復習できます。

## 学習範囲の位置づけ

現状は「中学数学のおさらい」を中心に、日本の高校数学I・数学Aの入口へつなぐ構成です。

| 範囲 | 現状のカバー | 補足 |
| --- | --- | --- |
| 中学数学 | かなりカバー | 数と式、平方根、文字式、方程式、関数、図形、データの基礎 |
| 数学I | 基礎範囲を拡張中 | 数と式、集合と命題、不等式、二次関数、三角比、データ分析 |
| 数学A | 基礎範囲を拡張中 | 場合の数と確率、図形の性質、数学と人間の活動の入口 |

数学I/Aを完全網羅するには、各単元の演習量、図形の証明、条件付き確率、期待値、数学と人間の活動の題材をさらに増やす必要があります。

## 今回厚めにしたところ

- 整数の範囲：正負の数、数直線、符号のルール、計算順序、数で見る分配法則
- ルートの入口：平方と平方根、平方数の因数を外へ出す整理、ルートの加減乗除
- 文字式への橋渡し：文字を「まだ決めていない数」として扱う、式の値、同類項、文字式の分配法則
- 方程式への接続：恒等式と方程式の違い、一次方程式のブロックモデル
- 数学I：集合と命題、不等式、二次関数の頂点と最大・最小、二次不等式、三角比、正弦定理・余弦定理、分散・標準偏差・相関
- 数学A：和の法則・積の法則、順列・組合せ、独立な試行、条件付き確率、期待値、図形の性質、整数と互除法の入口

## フォルダ構成

```text
.
├── index.html
├── assets
│   ├── css
│   │   └── styles.css
│   └── js
│       └── app.js
├── .nojekyll
└── README.md
```

ビルド手順は不要です。`index.html` をそのままブラウザで開けます。

数式表示には MathJax を使っています。GitHub Pages 上では CDN から読み込まれます。オフラインで開いた場合、数式は TeX 記法のまま表示されることがあります。

## ローカルで確認する

もっとも簡単な確認方法は、`index.html` をブラウザで開くことです。

ローカルサーバーで確認したい場合は、このフォルダで次を実行します。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開いてください。

## GitHub Pages で公開する

1. このフォルダの中身を GitHub リポジトリへ push します。
2. GitHub のリポジトリ画面で `Settings` → `Pages` を開きます。
3. `Build and deployment` の `Source` を `Deploy from a branch` にします。
4. `Branch` は `main`、フォルダは `/root` を選び、保存します。
5. 数十秒〜数分後に表示される Pages の URL へアクセスします。

このサイトは静的ファイルだけで動くため、GitHub Actions や npm build は不要です。

## 主な機能

- ページ送り式の単元ノート
- URL ハッシュによる単元への直接リンク
- 整数の数直線ラボ
- 分配法則の面積モデル
- 平方根の整理ラボ
- 同類項をまとめるラボ
- 一次方程式のブロックモデル
- 一次関数・二次関数グラフ
- 二次関数の頂点ラボ
- 三角比ラボ
- 確率の木ラボ
- 自動生成されるステップ問題
- 初期問題は2桁以内の計算を中心にし、計算量よりルール理解を優先
- 分野・レベルで絞り込める学習マップ
- MathJax による教科書寄りの数式表示

## 参照した範囲

数学I/Aの追加範囲は、文部科学省「高等学校学習指導要領（平成30年告示）」の数学I・数学Aの構成をもとにしています。

## 編集のヒント

- 単元を追加する場合は `assets/js/app.js` の `units` 配列へ追加します。
- 学習マップを追加する場合は `assets/js/app.js` の `topics` 配列へ追加します。
- 見た目を調整する場合は `assets/css/styles.css` を編集します。
