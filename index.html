整数から、ルートへ。ルートから、文字式へ。

計算ルールの「なぜそうなる？」を、手で動かして取り戻す。

正負の数、分配法則、平方根、同類項、方程式までを、短い説明・動く図解・自動生成問題で復習できます。 中学数学の土台から高校数学の入口まで、迷子になりやすい橋の部分を厚めにしました。

最初の単元から始める 問題で確認する
\(-3+5\) \(\sqrt{72}\) \(3(x-2)\)

今日の重点

  1. 符号は「向き」と「操作」で考える
  2. 平方根は「平方数を外へ出す」
  3. 文字式は「まだ決めていない数」の計算

式変形は暗記よりも、同じ操作を何度も見て触るのが効きます。

Learning path

最初に厚めにする計算ルールの流れ

数学が急に難しく見える原因の多くは、「数のルール」が「文字のルール」に変わったように見える瞬間です。 実際には、整数で使ったルールを、平方根や文字にも広げているだけです。

1

整数の範囲

正負、加減乗除、分配法則、計算順序を数直線と面積モデルで確認。

\(-2(3-5)=4\)
2

ルートが入る

平方根を「2乗すると戻る数」として見て、平方数の因数を外へ出す。

\(\sqrt{72}=\sqrt{36\cdot2}=6\sqrt{2}\)
3

文字式になる

まだ決まっていない数を箱として扱い、同類項・分配法則・式の値へ進む。

\(3x+2x=5x\)

Coverage

いまの学習範囲の見取り図

現状は「中学数学のおさらい」を中心に、数学I・数学Aの入口へつなぐ構成です。 数学I/Aを完全網羅と言い切るには、三角比・二次関数の最大最小・場合の数と確率・図形の性質をさらに厚くする必要があります。

中学数学

中心範囲：かなりカバー

  • 数と式：正負の数、平方根、文字式、方程式
  • 関数：比例・一次関数・二次関数の入口
  • 図形：基本量、相似、三平方の定理の入口
  • データ：平均・中央値・散らばりの入口

数学I

入口〜基礎：一部カバー

  • 数と式：実数・平方根・式の計算の橋渡し
  • 二次関数：グラフと係数の感覚
  • 図形と計量：現状は未展開、三角比を追加予定領域
  • データの分析：代表値と散らばりの入口

数学A

目次レベル：まだ薄い

  • 場合の数と確率：学習マップにはあるがレッスンは不足
  • 図形の性質：円・相似の入口のみ
  • 数学と人間の活動：未実装
  • 次の拡張で、問題生成と図解を足すとよい領域
