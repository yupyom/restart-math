知りたいところから、数学をおさらいできます

中学数学の土台から数学I・数学Aの入口までを、説明・動く図解・問題・読み物に分けています。 上から順に全部進める必要はありません。分からなくなった場所や、使いたい教材から選んでください。

使い方

  1. 説明を読む「単元」で言葉の意味と計算の理由を確認します。
  2. 目で確かめる「図解」を動かし、変わるもの・変わらないものを見ます。
  3. 自分で解く「問題」で途中の一手ずつを確認します。

迷ったときは「学習マップ」から、今の単元と前後のつながりを探せます。

学習範囲

いまの学習範囲の見取り図

現状は「中学数学のおさらい」を土台に、数学I・数学Aの基礎へ進める構成です。 数学I/Aの柱が見えるように、範囲ラベル・単元・動く図解・問題を追加し、数学B（数列）の入口も新設しています。

中学数学

中心範囲：かなりカバー

  • 数と式：正負の数、平方根、文字式、方程式
  • 関数：比例・一次関数・二次関数の入口
  • 図形：基本量、相似、三平方の定理の入口
  • データ：平均・中央値・散らばりの入口

数学I

基礎範囲：拡張中

  • 数と式：実数、集合と命題、不等式、式の計算
  • 二次関数：頂点、最大・最小、方程式・不等式との関係
  • 図形と計量：三角比、正弦定理・余弦定理の入口
  • データの分析：分散、標準偏差、散布図、相関の入口

数学A

基礎範囲：拡張中

  • 場合の数：和の法則、積の法則、順列、組合せ
  • 確率：独立な試行、条件付き確率、期待値の入口
  • 図形の性質：三角形、円、空間図形の入口
  • 数学と人間の活動：整数・互除法などへの導入

数学B（数列）

入口範囲：新設

  • 数列：等差数列・等比数列の定義と一般項
  • 和の計算：Σ記号の読み方、和の公式、階差数列
  • 漸化式：隣の関係から一般項を求める入口
  • 数学的帰納法：ドミノ倒しの証明法

単元ノート

ページ送りで進む単元ノート

学校の章順ではなく、次の説明で使う考えを先に学べる順に並べています。 文字式から方程式と文章題へ進み、図形の基本から三角比へ、代表値から分散・確率へ広げます。 URLのハッシュで単元を直接開けるので、必要な場所へ戻って復習できます。
